# Pocket Billiards Mode Finder

A sleek, modern web application inspired by Pocket Billiards, designed to demonstrate the calculation of the mode(s) in a dataset. This project features a professional UI with engaging visuals, suitable for impressing companies and fellow developers during interviews or presentations.

---

## Features

- User-friendly input for entering numbers separated by commas.
- Calculates the mode(s) of the provided dataset.
- Handles multiple modes gracefully.
- Fully responsive and visually themed around Pocket Billiards.
- Easy to extend or customize.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## Live Demo

Open the `index.html` file in your preferred web browser to try the application.

---

## How It Works

### Functional Overview
- Users input a comma-separated list of numbers.
- The JavaScript function `mode()` calculates the most frequently occurring value(s).
- Results are displayed dynamically on the page with an attractive billiards-themed design.

### Core Logic
- Counts the frequency of each number.
- Finds the maximum frequency.
- Returns all numbers that match this maximum frequency (handles multiple modes).

## Usage

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Enter numbers separated by commas into the input box (e.g., `1, 2, 2, 3, 3, 4`).
4. Click `Find Mode(s)` to see the result.
